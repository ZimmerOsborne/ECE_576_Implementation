Implementation of Statistical Analysis for Detection of Network Anomalies

This code is using signal analysis methods detailed in [1] to detect anomalies in network data. 
Datasets being used to test code are from the Canadian Institute for Cybersecurity at University of Newbrunswick [2] can be found at https://www.unb.ca/cic/datasets/ids-2017.html


[1] Barford et al., “A Signal Analysis of Network Traffic Anomalies,” Proc. ACM SIGCOMM Internet Measurement Workshop, 2002.
[2] Iman Sharafaldin, Arash Habibi Lashkari, and Ali A. Ghorbani, “Toward Generating a New Intrusion Detection Dataset and Intrusion Traffic Characterization”, 4th International Conference on Information Systems Security and Privacy (ICISSP), Portugal, January 2018.
